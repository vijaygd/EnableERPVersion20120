LEGEND:
OTHER ACTIVITIES YOU CAN DO ON THIS SCREEN:
An open Training Project can have the following status:
Assigned= Candidate(s) have been asigned to Assigned List of this Training Project.
Started= Atleast 1 candidate has been marked as 'actually started to attend training' in steps 3 or 3b.
Completed= All assigned candidates have been marked as 'Pass' or 'Fail' in the Final Status column.
After a Training Project has reached Completed status on the system, perform offline activity of updating attendance records, and then perform online activity of marking it as 'Closed'.
For closing a Training Project, hit the 'Close This Training Project' button in the top band. To delete candidates from the list, select candidates using check-box selection and click 'Delete Candidates'.
To add candidates to 'Candidate Calling', select candidates using check-box selection and click on 'Add to Candidate Calling' button.
To clear all details on this page for the given candidate, select candidate and use 'Reset' button.
To select all candidates for performing any given action, hit the 'Select All' button.
If any candidate has got a job, then you can update Job details about the candidate by selecting the candidate using check-box selection, and clicking on 'Add 'Got Job' Details' button. You can search for a given candidate in the below list by using Control+F function of the browser.
Click on Candidate name to view Profile.
Click on Candidate ID to view History.

LEGEND:
Candidate name in capitals indicates 'unemployed candidate'.
If candidate is already assigned to some employment project, then a $ appears before the candidate name.
If candidate is a priority candidate, then a * appears before candidate name.
If candidate is already assigned to some training program, then a # appears before the candidate name.
If candidate is a needy candidate, then a % appears before candidate name.
