OTHER ACTIVITIES YOU CAN DO ON THIS SCREEN:
Click on Candidate Name to view Profile.
Click on Candidate ID to view History.
You can search for a given candidate in the below list by using Control+F function of the browser.
To add candidates to 'Candidate Calling', select candidates using check-box selection and click on 'Add to Candidate Calling' button.
To delete candidates from the list, select candidates using check-box selection and click 'Delete Candidates'.
To re-set all details on this page for the given candidate, select candidate and use 'Reset' button.
To add notes for any candidate, then select candidate using check-box selection and click on 'Notes' button.
To select all candidates present on this page, use 'Select All' button.
LEGEND:
Candidate name in capitals indicates 'unemployed candidate'.
If candidate is already assigned to some other employment project, then a $ appears before the candidate name.
If candidate is a priority candidate, then a * appears before candidate name.
If candidate is already assigned to some training program, then a # appears before the candidate name.
If candidate is a needy candidate, then a % appears before candidate name.
